## fr-generator-hyx

对`fr-generator`的修改版本

## 安装

```bash
yarn add @clone-yangson/fr-generator-hyx
```

## 示例

```jsx
import 'antd/dist/antd.css';
import React from 'react';
import Generator from '@clone-yangson/fr-generator-hyx';

export default function Demo() {
  return <Generator />;
}
```
